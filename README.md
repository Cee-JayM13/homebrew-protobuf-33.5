# Cee-jaym13 Protobuf-33.5

## How do I install these formulae?

`brew install cee-jaym13/protobuf-33.5/<formula>`

Or `brew tap cee-jaym13/protobuf-33.5` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "cee-jaym13/protobuf-33.5"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
